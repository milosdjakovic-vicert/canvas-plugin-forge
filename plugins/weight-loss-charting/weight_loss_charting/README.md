# Weight Loss Charting Plugin

A Canvas plugin that adds a **Weight Loss Chart** button to note headers. Clicking it opens a comprehensive weight loss visit charting interface in a right chart pane.

## Features

- **Vitals Entry** — Weight, height, blood pressure, waist circumference with auto-BMI calculation
- **Weight Trend Chart** — Historical weight data visualized with Chart.js
- **Goal Tracking** — View existing goals and create new weight-related goals
- **Condition Assessment** — Assess active weight-related conditions (E66.* ICD-10)
- **Plan Narrative** — Structured plan with diet, exercise, medication, and follow-up sections

All commands are added as **uncommitted drafts** for clinician review before signing.
